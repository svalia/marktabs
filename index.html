Узнаёшь себя?

Ты хочешь работать с AI-ассистентом предметно, а не вручную собирать контекст перед каждым разговором?

  • Пять проектов — пять окон. Cmd+Tab по кругу, пока не найдёшь нужное?
  • AI прислал документ — а в редакторе это стена со звёздочками, решётками и обратными кавычками?
  • Копируешь куски из старых документов в новый чат, потому что ассистент не знает, что ты уже решил неделю назад?
  • VS Code — для кода, Obsidian — целый vault, а просто открыть несколько .md — нечем?

Как это работает

  1. Открой файлы

    Через File > Open или клик в sidebar — все появляются как вкладки в одном окне. Вместо множества окон с открытыми .md файлами — одно окно с удобной навигацией по вкладкам.

  2. Читай в чистом виде

    Cmd+R — документ рендерится мгновенно: заголовки, таблицы, код, ссылки кликабельны. Читаешь как текст, не продираешься сквозь синтаксис.

  3. Правь без смены инструмента

    Cmd+R снова — вернулся в edit mode. Cmd+S сохраняет в любом режиме. Всё в одном окне, ничего не теряешь.

К чему ты придёшь

  • Не будешь тратить мыслетопливо на управление окнами — только на содержание документов
  • Рабочий контекст под контролем, а не разбросан по пяти местам
  • Перестанешь раздражаться каждый раз, когда AI присылает документ с синтаксисом
  • Все открытые AI-проекты видны сразу — переключение за одно действие
  • Накопленный контекст лежит в .md файлах, структурированно, рядом

Есть сомнения?

"Ещё один редактор, буду разбираться неделю"
Установка занимает минуту. Нет аккаунта, нет vault, нет миграции. Перетащи файл, открой папку — всё.
"Бесплатно, без App Store — что-то не то"
Это целевое решение, не демо. Распространение через DMG с лендинга — быстрее и без ограничений App Store. Никакого облака, никаких данных на серверах.
"Разве VS Code не справляется?"
Справляется, но это инструмент для кода. Он не даёт читать документ в чистом виде, extensions для Markdown — сторонние, интерфейс заточен под разработку. MarkTabs — для документов. Нет лишнего.

Почему не VS Code, Obsidian или TextEdit

VS Code
Перегружен для документов: панели, extensions, developer-контекст. MarkTabs — только .md, только работа с текстом.
Obsidian
Vault-модель, синхронизация, плагины. Если нужна база знаний — Obsidian. Если нужно открыть несколько файлов и работать — MarkTabs.
TextEdit
Нет вкладок, нет sidebar, нет Markdown рендеринга. MarkTabs закрывает нишу, которую macOS оставила пустой.
